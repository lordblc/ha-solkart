# Solkart

Solar production forecast sensors from the Norwegian [Solkart](https://solkart.no/api)
API — power now, energy today/tomorrow, peak time, GHI and per-array power.
Multiple panel arrays in a single request, Energy-dashboard support, and a free-tier
friendly default polling interval.

Add your free API key under **Settings → Devices & Services → Add Integration →
Solkart**. See the README for full configuration and sensor details.
