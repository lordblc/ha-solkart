# Brands submission (optional)

These files are pre-arranged in the exact layout the
[home-assistant/brands](https://github.com/home-assistant/brands) repository
expects, so the integration shows its logo in Home Assistant / HACS instead of a
default placeholder.

```
custom_integrations/solkart/icon.png      (256x256)
custom_integrations/solkart/icon@2x.png   (512x512)
```

## How to submit

1. Fork <https://github.com/home-assistant/brands>.
2. Copy the `custom_integrations/` folder from here into the root of your fork.
3. Open a pull request.

Once merged, you can drop `ignore: brands` from `.github/workflows/validate.yaml`.

The icons are generated by [`../icons/make_icon.py`](../icons/make_icon.py) — edit
that script and re-run it to regenerate every size, including these.
